REFACTOR CHANGELOG — branch: refactor/clique-topology
================================================================================
Tracking all changes made in this refactor session.
Each entry: [STEP N] short description, then details.

Theoretical motivation:
  - Switch from Vietoris-Rips with distance-ε filtration to clique topology
    with edge density ρ as filtration parameter (Giusti et al. 2015)
  - ρ = k / C(N,2) where k = edges added, C(N,2) = N(N-1)/2 = 153 for 18ch
  - Compute PH per epoch, then average Betti curves / persistent landscapes
    (not averaging connectivity matrices, which discards topological structure)
  - Persistent landscapes enable statistically sound averaging across epochs
  - Scope: correlation-only by default; wPLI and coherence opt-in via config

References:
  - Giusti et al. 2015 — 1502.06172v1.pdf (clique topology, order complex)
  - Curto & Sanderson 2025 — annurev-neuro-112723-034315.pdf (TDA in neuro)

[x] STEP 3 — Persistent landscapes
      New function _compute_landscape(dgm_rho, rho_values, n_levels):
        - Builds tent function for each (birth_ρ, death_ρ) pair
        - λ_k(t) = k-th largest tent value at each ρ step
        - Returns (n_levels, N_FILT_STEPS) array
      _run_tda_pipeline extended: computes landscape per epoch in ρ space,
        sums and averages across epochs alongside Betti curves.
      landscape_auc (AUC of λ_1) added to features dict for H0 and H1.
      _extract_betti_features: surfaces landscape_auc_b0/b1 and full
        landscape arrays (landscape_0, landscape_1) for visualisation.
      config.py: added N_LANDSCAPE_LEVELS = 5.
      Completed: 2026-04-06

[x] STEP 4 — Wire config flags into pipeline_subject.py
      INTERPOLATE_BAD_CHANNELS:
        _preprocess: RANSAC runs for diagnostics only when False.
        Rationale: RANSAC may mistake genuine ADHD signal for noise
        (atypical spatial patterns → systematic preprocessing bias vs TDC).
        When False: info['bads'] cleared after RANSAC — all channels treated
        as good, avg reference and ICA see the full unmodified channel set.
        When True: original interpolation behaviour preserved.
      COMPUTE_NO_SPATIAL_FILTER:
        Steps 2-5 (connectivity, TDA, betti features, graph theory) for the
        no-spatial-filter pipeline are now gated on this flag. If False,
        all those steps are skipped with a log message. Figures block also
        guarded against None dist_per_epoch.
      COMPUTE_COHERENCE, COMPUTE_WPLI, COMPUTE_SURFACE_LAPLACIAN:
        Already wired in Step 2 — no changes needed here.
      Completed: 2026-04-06

[x] STEP 5 (null model) — Epoch-level matrix shuffling null model
      Removed N_SURROGATES / SURROGATE_PERCENTILE from config.py.
      Added N_SHUFFLES = 200 to config.py.
      _run_tda_pipeline extended: for each epoch, alongside real PH,
        shuffle the 171 off-diagonal dist values K=200 times, run ripser
        on each → K null Betti curves per epoch. Accumulate:
          - null_sum: running sum of per-epoch mean null curves
          - p_sum: per-epoch fraction of shuffled AUCs ≤ real AUC
        After epoch loop: mean_null = null_sum / valid_epochs
                          p_value  = p_sum  / valid_epochs
      p_value stored in features dict per dimension (B0, B1).
      null_betti stored in tda_results[key] for visualisation.
      _extract_betti_features: surfaces p_value_b0/b1, null_betti_0/1.
      Note: TDADHD_copy.py phase-randomization surrogates left intact
      (prototype file — kept as-is per project brief).
      Completed: 2026-04-06
